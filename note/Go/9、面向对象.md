## 继承

​		一个struct嵌套了另一个匿名结构体，那么这个结构体可以直接访问匿名结构体的字段和方法，从而实现了继承特性。

```go
type Goods struct {
	Name string
	Price int
}

type Book struct {
	Goods
	Writer string
}
```
